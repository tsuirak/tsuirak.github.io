文件结构

package.json

包含名称、描述和版本之类的信息，以及运行、开发以及有选择地将项目发布到 NPM 所需的信息。是项目的核心配置。npm in stall，将会执行安装配置。

➡ 详细了解

main.js

主程序入口。

config

dev 和 prod 指定访问的接口路径，两者区别在于运行的环境（npm run dev/prod）；

index 可以指定端口号，eslint 等。

需要在 index.js 中修改 useEslint：false（语法校验！）

src

api：定义接口；

assets：相关静态资源；

components：相关的第三方组件；

icons：图表；

router：路由文件；

store：相关 js 文件；

styles：相关 css 文件；

utils：相关工具类；

views：核心的页面。
